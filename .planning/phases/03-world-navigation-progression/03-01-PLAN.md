2. **新增状态:**
   - `worlds` state: `useState<ArenaWorldInfo[]>([])`
   - `progress` state: `useState<Record<string, LevelProgress>>({})`
   - `loading` state: `useState(true)`
   - 保留现有 `viewState`, `selectedLevel`, `arenaSSE` 不变
   - 删除 `showRunControls` state（不再需要单独的 RunControls 全屏显示）

3. **新增数据获取 useEffect:**
   ```typescript
   useEffect(() => {
     let cancelled = false;
     setLoading(true);
     authFetch('/api/arena/levels')
       .then(res => { if (!res.ok) throw new Error(`HTTP ${res.status}`); return res.json(); })
       .then((data: { worlds: ArenaWorldInfo[] }) => {
         if (!cancelled) setWorlds(data.worlds ?? []);
       })
       .catch(() => {}) // 静默处理，树显示空
       .finally(() => { if (!cancelled) setLoading(false); });
     return () => { cancelled = true; };
   }, []);
   ```

4. **新增进度刷新函数 `refreshProgress`:**
   ```typescript
   const refreshProgress = useCallback(() => {
     authFetch('/api/arena/progress')
       .then(res => res.ok ? res.json() : { progress: {} })
       .then((data: { progress: Record<string, LevelProgress> }) => {
         setProgress(data.progress ?? {});
       })
       .catch(() => {});
   }, []);
   ```
   在 worlds 加载完成后调用 refreshProgress
   在 arenaSSE.status 变为 'completed' 时也调用 refreshProgress（运行完成后刷新进度）

5. **修改 handleSelectLevel:**
   ```typescript
   const handleSelectLevel = useCallback((worldId: string, level: ArenaLevelInfo) => {
     setSelectedLevel({ worldId, level });
     // 不切换 viewState — 双栏模式下选中关卡只影响右侧内容
     if (viewState.phase !== 'running') {
       setViewState({ phase: 'levels' });
     }
   }, [viewState.phase]);
   ```

6. **新增 isRunning 判断:**
   ```typescript
   const isRunning = viewState.phase === 'running';
   ```

7. **渲染结构改为双栏 (per D-01, D-02):**

   return (
